# Singlish Converter (සිංහල පරිවර්තකය)

මේක සරල සහ පහසු Singlish Converter එකක්. ඔයාට Singlish වලින් (ඉංග්‍රීසි අකුරු වලින්) සිංහල වචන ටයිප් කරන්න පුළුවන්.

## විශේෂාංග

1.  **Unicode Converter (වෙබ් අඩවි සඳහා):**
    *   Facebook, WhatsApp, හෝ වෙබ් අඩවි වල පාවිච්චි කරන්න පුළුවන් සිංහල අකුරු (Unicode) ලබාගන්න පුළුවන්.
    *   උදාහරණ: "mama" -> "මම"

2.  **Legacy Converter (Photoshop/CapCut සඳහා):**
    *   Photoshop, Premiere Pro, CapCut වගේ මෘදුකාංග වල පාවිච්චි කරන්න පුළුවන් සිංහල අකුරු (Legacy) ලබාගන්න පුළුවන්.
    *   මේ සඳහා **FM Abhaya** හෝ **DL-Paras** වගේ ෆොන්ට් එකක් ඔයාගේ පරිගණකයේ තියෙන්න ඕනේ.

3.  **Typing Cheat Sheet (අකුරු වගුව):**
    *   අමාරු අකුරු ගහන විදිය බලාගන්න පුළුවන් වගුවක් ඇතුළත් කරලා තියෙනවා.

## පාවිච්චි කරන විදිය

1.  **Type here (Singlish)** කොටුවේ ඔයාට ඕන දේ Singlish වලින් ටයිප් කරන්න.
    *   උදාහරණ: "mama gedara yanawa"
2.  **Unicode Output** කොටුවෙන් වෙබ් අඩවි වලට ඕන සිංහල අකුරු කොපි කරගන්න.
3.  **Legacy Output** කොටුවෙන් Photoshop/CapCut වලට ඕන අකුරු කොපි කරගන්න.
    *   Legacy අකුරු පේස්ට් කළාට පස්සේ, අදාළ මෘදුකාංගයේ ෆොන්ට් එක **FM Abhaya** වලට මාරු කරන්න.

## තාක්ෂණික තොරතුරු

*   **Framework:** React 19
*   **Styling:** Tailwind CSS
*   **Build Tool:** Vite
*   **Icons:** Lucide React
*   **Animations:** Motion (Framer Motion)

## සංවර්ධනය (Development)

මේක ඔයාගේ මැෂින් එකේ රන් කරන්න:

```bash
npm install
npm run dev
```
